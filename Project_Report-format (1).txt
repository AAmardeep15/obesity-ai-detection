ABSTRACT

This project presents an end-to-end machine learning system for obesity risk classification and personalized health recommendation. The solution is developed as a Flask-based web platform that performs local model inference without external AI APIs. It predicts one among seven obesity classes using demographic, lifestyle, and anthropometric features.

The ML pipeline includes preprocessing, outlier handling, feature engineering (BMI), scaling, and ensemble learning. Three base models (Random Forest, Logistic Regression, Gradient Boosting) are trained and combined through soft voting. On the current dataset split, the ensemble achieved 84.60% accuracy and 84.58% weighted F1 score.

Beyond classification, the system generates class-aligned nutrition and exercise plans and supports downloadable report generation. The project demonstrates practical integration of data science and software engineering for preventive healthcare support.

Keywords: Obesity Classification, Ensemble Learning, Personalized Nutrition, Exercise Recommendation, Flask, Scikit-learn, BMI

CHAPTER 1
INTRODUCTION

Obesity is one of the fastest growing public health concerns worldwide. Early risk identification and timely behavior correction can significantly reduce long-term complications such as type-2 diabetes, cardiovascular disease, fatty liver, and joint disorders. However, most people do not have access to personalized and immediate obesity risk feedback in day-to-day life.

This project addresses that gap through a practical web-based prediction system that combines machine learning with personalized guidance. The application takes user inputs, predicts obesity class, and generates tailored nutrition and exercise recommendations. The system is designed for educational and decision-support usage and keeps ML inference fully local.

The project aims to deliver a balanced solution across three dimensions: prediction quality, user experience, and transparency. We also provide model statistics, confusion matrix, and feature importance for interpretability.

CHAPTER 2
BASIC CONCEPTS / LITERATURE REVIEW

2.1 Obesity and BMI

Body Mass Index (BMI) is a standard screening metric:
BMI = Weight (kg) / Height^2 (m^2)

Although BMI is not a complete diagnostic metric, it is effective for large-scale screening and class-level risk modeling.

2.2 Supervised Machine Learning for Health Classification

The project uses supervised multi-class classification. Historical labeled examples are used to learn decision boundaries for seven obesity classes:
- Insufficient_Weight
- Normal_Weight
- Overweight_Level_I
- Overweight_Level_II
- Obesity_Type_I
- Obesity_Type_II
- Obesity_Type_III

2.3 Models Used

a) Random Forest Classifier
- Ensemble of decision trees with bagging
- Good non-linear modeling and robust performance

b) Logistic Regression
- Strong baseline linear classifier
- Useful for comparability and calibration

c) Gradient Boosting Classifier
- Sequential weak learners that correct residual errors
- Effective for structured tabular data

d) Soft Voting Ensemble
- Combines class probabilities from all base models
- Produces improved generalized prediction quality

2.4 Data Preprocessing Concepts

- Missing value handling (mean/mode)
- Outlier capping using IQR bounds
- Label encoding for categorical variables
- Train-test split with stratification
- Standard scaling fit only on training split to avoid leakage

2.5 Recommendation Layer

The recommendation subsystem maps predicted class to:
- Nutrition plan (calorie target, meals, avoid list, health tips)
- Exercise plan (goal, weekly target, exercise cards, avoid list, tips)

This layer is deterministic and supports fast response time.
